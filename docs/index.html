Metrics

What Brain Battery Measures

Four distinct signals, each grounded in published neuroscience research. Not emotional stress — cognitive workload is a different construct entirely.

⚡
Cognitive Workload
Probability of high mental demand, estimated by a three-tower multimodal neural network fusing EEG, physiological, and spectral features.
P(high CW) = softmax(workload_head(EEG ⊕ Physio ⊕ Freq))₁
〜
Mental Fatigue
EEG spectral ratio between theta and alpha bands. Rising theta relative to alpha indicates cognitive fatigue — no model involved, pure signal processing.
log θ_power − log α_power   (Klimesch, 1999)
◈
Focus Streak
Timer counting continuous low-workload windows. A grace period of three consecutive high-load windows is required before the streak resets — resists momentary spikes.
3-window grace period before reset
◎
Signal Quality
Per-channel variance heuristic in live mode; spectral attention weight proxy in demo mode. Used to weight modality contributions inside the fusion layer.
σ heuristic · Viola et al. (2009)
Workflow

How It Works

Two modes: replay pre-recorded subjects, or stream live from a Muse headband.

01

Choose a mode

Demo mode replays 12 pre-recorded subjects from the UNIVERSE dataset with adjustable playback speed, subject selector, and full physio + EEG waveforms.

Live mode connects to a Muse headband via BlueMuse (Windows LSL bridge) for real-time inference. Live mode uses EEG only — Muse does not capture physiological signals.

02

Run a session

The model processes 1-second EEG windows in real time, updating the workload gauge, fatigue index, and focus streak with each frame. Session data is logged automatically.

03

Review your history

The History tab shows all past sessions as a bar chart with per-session averages for workload, fatigue, and focus time. Each session is stored locally in history.jsonl.

04

Personalization (Live mode)

An online Bayesian calibration layer adapts the workload threshold to your individual cognitive baseline across sessions. Zero-calibration (population prior) → calibrating → personalized — the model gets better as it learns you.

Model

Architecture

A three-tower multimodal network with learned attention fusion, trained with Leave-One-Subject-Out cross-validation on 12 subjects. 

  EEG     (4 ch × 256 samples)   →   EEGNet  (depthwise/separable CNN)   ─┐
  Physio  (BVP + HR + EDA)        →   1D-CNN  (3-layer)                   ├→  Attention Fusion  →  Workload Head  →  P(high CW)
  Spectral (36 features)           →   MLP     (with BatchNorm + Dropout)  ─┘

  Attention Fusion:  learned per-modality weights conditioned on signal quality index
  Workload Head:     Linear(FUSION_DIM, 128) → BN → ReLU → Dropout → Linear → Softmax
  Labels:            NASA-TLX binary (high / low workload)
  Validation:        LOSO  (12 folds, one subject held out per fold)
EEGNet
Compact depthwise-separable CNN purpose-built for EEG. Fewer parameters than generic CNNs while capturing temporal and spatial features across channels.
Lawhern et al. (2018)
Attention Fusion
Signal quality index (SQI) for each modality is passed alongside tower embeddings. The fusion layer learns to down-weight noisy or missing modalities.
Conditioned on SQI vector
LOSO Validation
Each of the 12 folds holds out one subject completely. This measures how well spectral features generalize to unseen individuals — the hardest generalization test.
n=12 subjects · zero data leakage

Installation

Get Started

Runs locally. No cloud account needed for Demo mode — just Python and the dataset.

1 — Clone and install
# clone the repo
git clone https://github.com/NotAverageGitHubUser/Brain-Battery.git
cd Brain-Battery

# install dependencies
pip install streamlit torch numpy scipy plotly kaggle
2 — Launch (dataset downloads automatically on first run)
python -m streamlit run dashboard.py
On first launch, Brain Battery will show a setup screen. Enter your Kaggle API credentials and click Download Dataset — the ~300 MB dataset downloads and unzips automatically.
3 — Live mode (optional — Muse headband required)
# install BlueMuse (Windows), start LSL stream, then connect in the app
https://github.com/kowalej/BlueMuse
