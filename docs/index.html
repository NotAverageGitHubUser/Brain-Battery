Metrics

What Brain Battery Measures

Four distinct signals grounded in published neuroscience research. Cognitive workload is not emotional stress — different construct, different labels, different biomarkers.

01 — Primary Metric
Cognitive Workload
Probability of high mental demand, estimated by a three-tower multimodal neural network fusing EEG, physiological signals, and spectral features. Trained on NASA Task Load Index labels with Leave-One-Subject-Out cross-validation across 24 subjects.
P(high CW) = softmax(workload_head(EEG ⊕ Physio ⊕ Freq))₁
02 — Spectral Ratio
Mental Fatigue
EEG theta-to-alpha power ratio. Rising theta (4–8 Hz) relative to alpha (8–13 Hz) indicates cognitive fatigue building over time. Pure signal processing — no model involved. Validated across decades of neuroscience literature.
log θ_power − log α_power  (Klimesch, 1999)
03 — Sustained Attention
Focus Streak
Timer tracking continuous low-workload windows. Resistant to momentary spikes — the streak only resets after three consecutive high-load windows, preventing false interruptions from brief task transitions.
Reset threshold: 3 consecutive high-CW windows
04 — Confidence Weighting
Signal Quality
Per-channel standard deviation heuristic in live mode; spectral attention weight proxy in demo mode. The signal quality index (SQI) is passed into the fusion layer so noisy or poorly-connected channels are automatically down-weighted.
σ heuristic per channel  (Viola et al., 2009)
Model

Architecture

Three-tower multimodal network with learned attention fusion, trained with Leave-One-Subject-Out cross-validation across 24 subjects. 

  EEG      (4 ch × 256 samples)   →   EEGNet  (depthwise/separable CNN)     ─┐
  Physio   (BVP + HR + EDA)        →   1D-CNN  (3-layer conv)                ├→  Attention Fusion  →  Workload Head  →  P(high CW)
  Spectral  (36 features)           →   MLP     (BatchNorm + Dropout)         ─┘

  Fusion:   learned per-modality weights conditioned on signal quality index (SQI)
  Head:     Linear → BN → ReLU → Dropout → Linear → Softmax(2)
  Labels:   NASA-TLX binary  (high / low workload)
  Val:      LOSO  ·  24 subjects  ·  zero data leakage across folds
EEGNet
Compact depthwise-separable CNN purpose-built for EEG. Captures temporal and spatial features with far fewer parameters than generic CNNs.
Lawhern et al. (2018)
Attention Fusion
SQI vector conditions the fusion layer to down-weight noisy or absent modalities. Enables robust inference when physio signals are unavailable (live mode).
SQI-conditioned weighting
LOSO Validation
Each fold holds out one subject entirely — the hardest generalization test. Measures whether spectral features transfer to people the model has never seen.
24 subjects · zero leakage
Workflow

How It Works

Two modes: replay pre-recorded subjects from the UNIVERSE dataset, or stream live from a Muse headband.

01

Choose a mode

Demo mode replays 12 pre-recorded subjects from the UNIVERSE dataset with adjustable playback speed, subject selector, and synchronized EEG + physio waveforms.

Live mode connects to a Muse S or Muse 2 headband via BlueMuse (Windows LSL bridge) for real-time EEG inference. Live mode uses EEG only — Muse does not record physiological signals.

02

Run a session

The model processes 1-second EEG windows continuously, updating the cognitive workload gauge, fatigue index, and focus streak timer on each frame. Session data is appended to a local log file automatically.

03

Review your history

The History tab shows all past sessions as a bar chart with per-session averages for workload, fatigue, and focus time. Each session is stored in history.jsonl and persists across restarts.

04

Personalization (Live mode)

An online Bayesian calibration layer adapts the workload threshold to your individual cognitive baseline across sessions. Three stages: zero-calibration (population prior) → calibrating → personalized. The model improves as it accumulates data from you.

Installation

Get Started

Runs locally. No cloud setup needed for Demo mode — just Python and the dataset.

1 — Clone and install
# clone the repo
git clone https://github.com/NotAverageGitHubUser/Brain-Battery.git
cd Brain-Battery

# install dependencies
pip install streamlit torch numpy scipy plotly kaggle
2 — Run (dataset setup is automatic on first launch)
python -m streamlit run dashboard.py
On first launch, Brain Battery shows a setup screen. Enter your Kaggle API credentials and click Download Dataset — the ~300 MB dataset downloads and unzips automatically. No manual file management needed.
3 — Live mode (optional — Muse headband + BlueMuse required)
# Windows: install BlueMuse, start the LSL stream, then connect inside the app
https://github.com/kowalej/BlueMuse

Research Background

The dataset and model are grounded in peer-reviewed neuroscience and BCI research.

UNIVERSE: A Multimodal Dataset for Cognitive Workload Estimation

The original UNIVERSE dataset paper describing the 24-subject study, recording protocol, EEG + physiological signal collection, and NASA-TLX labeling methodology. This is the primary data source for Brain Battery's Demo mode.
